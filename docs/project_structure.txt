directories

- config -> text files containing experimental parameters

- data
---- archive -> archived experimental results
---- temp -> hold experimental instance sets
---- template -> templates for instance generate and summary sheets

- docs -> documentation for project

- src
---- common
------- instance generation -> generate bootstrap samples of instances
------- model helpers -> common classes used between all models
------- solution checker -> check validity of solution against constraints
---- models
------- basic_cp_pack -> CP model for packing
------- basic_cp_sched -> CP model for scheduling
------- basic_edd_pack -> greedy EDD based heuristic for packing
------- basic_lbbd -> LBBD using CP model for packing and CP model for scheduling
------- basic_mip_pack -> MIP model for packing
------- cluster_pack -> size-constrained clustering using KmeansS
------- edd_parallel_sched -> parallel SGS for GRCPSP
------- edd_serial_sched -> serial SGS for GRCPSP
------- genetic_sched -> genetic algorithm based on Hartmann (1998)
------- relaxed_cp_sched -> relaxation of basic_cp_sched model
